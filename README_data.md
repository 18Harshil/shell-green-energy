# data/

Place the EMHIRES CSV files here before running the notebook.

## Files needed

| File | Source |
|---|---|
| `EMHIRESPV_NUTS2_level.csv` | Download from Kaggle (see main README) |
| `EMHIRES_WIND_COUNTRY_June2019.csv` | Download from Kaggle (see main README) |

**Kaggle link:** https://www.kaggle.com/datasets/pythonafroz/wind-and-solar-power-generation-data

> If files are missing, the notebook will automatically generate synthetic data
> with the same EMHIRES structure so you can still run every cell.
